# prim
### Overview
Simple crate to find prime numbers.
Source code can be found on [Github](https://github.com/GitKat64/prim).

### Example
```
extern crate prim;

fn main() {
    println!("{}", prim::check_prime(5)); //true
    println!("{:?}", prim::check_area(1, 10)); //[1, 2, 3, 5, 7]
}
```
